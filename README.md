# Abyss - Roguelike Underwater Adventure

Abyss is a roguelike game about diving deep underwater and discovering mysterious sea creatures. Each dive is unique, with procedurally generated environments, creatures, and treasures.

## Features

- **Procedural Dungeon Generation**: Each dive generates a unique underwater cavern
- **Sea Creatures**: Encounter diverse underwater creatures with unique behaviors
- **Roguelike Mechanics**: Permadeath, progression between runs, upgrades
- **Underwater Exploration**: Dive deeper to find rarer creatures and treasures
- **Dynamic Progression**: Unlock new creatures and abilities as you progress

## Project Structure

```
Assets/
├── Scripts/
│   ├── Core/              # Core game logic
│   ├── Creatures/         # Sea creature definitions and behaviors
│   ├── Dungeon/           # Procedural generation and level management
│   ├── Player/            # Player character and mechanics
│   ├── UI/                # User interface
│   └── Utilities/         # Helper utilities
├── Prefabs/               # Prefab templates
├── Scenes/                # Game scenes
├── Art/                   # Sprites, models, animations
└── Audio/                 # Sound effects and music
```

## Getting Started

1. Clone this repository
2. Open the project in Unity (2022 LTS or later recommended)
3. Open the `MainScene` under `Assets/Scenes/`
4. Press Play to start developing!
